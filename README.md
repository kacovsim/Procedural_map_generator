# Procedural_map_generator
[► View website](https://mapgenerator.neocities.org/)
            <div>
              This project is an interactive, web-based Procedural Map Generator that dynamically creates 2D terrain
              maps. It provides a user-friendly interface where users can tweak algorithmic parameters and color
              schemes
              to generate unique landscapes. <br><br>
              <ul>
                <li><b>Algorithmic Generation:</b> The core of the generator utilizes the Diamond-Square algorithm to
                  calculate and build detailed height maps.</li>
                <li><b>Customizable Terrain:</b> Users can fine-tune the landscape by adjusting the map's size. The
                  tool
                  also allows for adjusting roughness, randomness, and starting
                  corner values to influence the terrain's generation, such as forcing rivers to form.</li>
                <li><b>Elevation-Based Coloring:</b> The visual output is completely customizable, allowing users to
                  use
                  color pickers and height inputs to define specific color schemes across different elevation levels.
                </li>
                <li><b>Thematic Presets:</b>The generator includes several quick-styling preset options inspired by
                  popular themes and gives users the ability to save up to three
                  custom presets.</li>
                <li><b>Responsive Canvas Rendering:</b>Maps are rendered instantly on an HTML5 canvas, wrapped in a
                  responsive, flexbox-driven design that adapts to different screen sizes and aspect ratios.</li>
              </ul>
              <br>
            </div>
